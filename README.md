# Global Airbnb Performance Dashboard

A Power BI dashboard analyzing Airbnb listings, reviews, ratings, pricing, trust metrics, and seasonality trends across 10 global cities.

## Dataset

Dataset Source:
https://mavenanalytics.io/data-playground/airbnb-listings-reviews

## Key Insights

- Paris, New York, and Sydney account for nearly 48% of all listings.
- Non-Superhost properties dominate across all cities.
- Mexico City has the highest overall rating, while Hong Kong has the lowest.
- Hotel rooms have the highest average price; private rooms are the most affordable.
- 86.5% of reviewers leave only one review, indicating a highly concentrated reviewer base.
- 66.9% of users are both verified and have profile pictures.
- Airbnb listings grew rapidly until 2015–2016 before declining, with a significant drop during 2020–2021.
